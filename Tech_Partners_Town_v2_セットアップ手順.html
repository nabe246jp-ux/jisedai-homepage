SETUP GUIDE

Tech Partners Town v2 立ち上げ手順

Next.js + Three.js を使った自律型ホームページのセットアップ。
非エンジニアでも順番に進めれば必ず動きます。

1

Claude Code を起動して、プロジェクトフォルダに移動する

スタートメニューから「Claude Code」を起動して、下のコマンドをコピーして貼り付けて実行してください。

cd "C:\Users\nabe2\Documents\Claude\Projects\次世代ホームページ\tech-partners-town-v2"
補足: cd は「change directory」の略で、作業フォルダを変える命令です。 成功するとプロンプトの表示にフォルダ名が出ます。
2

必要なライブラリをインストールする（初回のみ・約2〜5分）

Next.js や Three.js などの部品をまとめて取り寄せます。

npm install
補足: 大量の進捗表示が流れますが、エラーで止まらなければ正常です。
最後に「added xxx packages」と出れば成功。
トラブル時: 「npm: command not found」と出た場合は、Node.js（バージョン18以上）がインストールされていません。 https://nodejs.org/ja から LTS 版をダウンロード・インストールしてください。
3

開発サーバを起動する

町を実際に立ち上げます。

npm run dev
補足: 「Ready」「Local: http://localhost:3000」と表示されたら準備完了。
4

ブラウザで町を開く

普段使っているブラウザで下のアドレスにアクセスします。

http://localhost:3000

初回は10〜20秒ほど読み込みがかかります。「BUILDING THE TOWN…」と表示されたあと、町が現れます。

5

町でできること（操作ガイド）

マウスを動かす — カメラがゆっくり追従し、町を見渡せます
左下の時刻パネル — 実時刻が秒単位で進む。季節と時間帯のラベルも表示
右上のKPIボード — 売上・利益・社員数・訪問者・稼働中AIが25秒ごとに更新。「UPDATE」を押すと即更新＋ビルが光る
左下のデモ操作 — 「＋3時間」で空が変わる、「＋1ヶ月」で季節装飾が変わるのを即確認
右下の妖精ルミナ — クリックして話しかけると、テンション高めに答えます
6

サーバを止めるとき

Claude Code のターミナル上で、下のキーを同時押し。

Ctrl + C

もう一度立ち上げるときは、ステップ1（cd）→ ステップ3（npm run dev）でOK。npm install は最初の1回だけです。

7

もし動かなかったら

真っ白なまま — F12キー（またはCtrl+Shift+I）で開発者ツールを開き、Console タブのエラー文を Claude に貼り付けて相談
「Module not found」のエラー — npm install が失敗した可能性。再度実行
カクつく — ブラウザを Chrome か Edge にしてみる。それでも重い場合は伝えてください、軽量化版を作ります
