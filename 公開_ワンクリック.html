Paste Once

PowerShellに1行だけ貼り付ければ公開できます。

前回の方法はWindowsの文字コードの相性で動きませんでした。今回は .ps1 ファイルを使わず、PowerShellに長い1行をそのまま貼り付ける方式に変えました。これなら文字コードの問題は起きません。

1このコマンドをコピー

右側の青い「コピー」ボタンをクリックします。

cd 'C:\Users\nabe2\Documents\Claude\Projects\次世代ホームページ\smart-site-mvp'; New-Item -ItemType Directory -Force -Path .\public | Out-Null; Copy-Item '..\VantageWorks_SmartSite_preview.html' '.\public\vantageworks.html' -Force; git add public/vantageworks.html; git commit -m "publish vantageworks preview"; git push origin main

2PowerShell に貼り付け → Enter

すでに開いているPowerShellの中で、右クリック（または Ctrl + V）で貼り付け、Enter を押します。

PowerShellが閉じてしまっている場合は、Windowsキー → 「powershell」と入力 → Enter で開けます。

31〜2分待って完了

「git に追加」「コミット」「push」が順に表示されて、最後にコマンドプロンプトに戻れば成功です。GitHubに送られて、Vercelが自動でWebに公開します。
