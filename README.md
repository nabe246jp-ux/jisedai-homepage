# 次世代ホームページ プロジェクト

「モックのように動くホームページ」をテーマに、企業体験・商品体験ができる次世代型のWebページの見本(モック)を集めたプロジェクトです。

## 内容

`index.html` を開くと、3つのモックを切り替えられるトップページが表示されます。

### Mock 01 — TECH PARTNERS の街
ファイル: `mock_v1_techpartners_town.html`

架空のIT会社 TECH PARTNERS の街並みを、マウスを動かして歩き回れるホームページ。建物は4つあり、それぞれ「研修について」「事業について」「福利厚生について」「会社概要」を表現しています。建物をクリックすると、その建物の中(カード一覧)に入れます。各カードをクリックするとさらに詳細情報が展開されます。

主な特徴
- マウス追従の「あなた」キャラクターが道なりに歩く
- 建物に近づくとふわっと光る
- 各建物に8枚(または6枚)のカード、合計30枚のカードがすべて詳細展開可能
- 戻り方は4種類(Esc / 右クリック / 背景クリック / 戻るボタン)
- 噴水、教会、ストリングライト、屋台、人々の動きで街を演出

### Mock 02 — KOTONOHA 万年筆工房
ファイル: `mock_v2_kotonoha_pen_showroom.html`

架空の万年筆店のショールームを歩いて回るホームページ。落ち着いた高級感のある雰囲気で、ガラスケースに並んだ5本の万年筆をクリックすると拡大して書き味を見られます。

### Mock 03 — TECH PARTNERS 採用
ファイル: `mock_v3_recruit_office.html`

架空のIT会社の採用情報を、ビルの断面図として表現したページ。各階(事業・福利厚生・研修・募集)を歩いて、項目をクリックして詳細を3段階まで深掘りできます。

## 動かし方

各 HTML ファイルをブラウザで開くだけで動きます。インターネット接続は不要です。`index.html` から好きなモックを選んで開くか、それぞれの HTML ファイルを直接ダブルクリックしてください。

## ライセンス

このプロジェクトは見本(デモ)用です。会社名・代表者名・数字などは架空のものです。
