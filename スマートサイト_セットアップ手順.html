SMART SITE — Setup Guide

スマートサイトMVPを
あなたのパソコンで動かす手順

このページの通りにコピペで進めれば、ブラウザで実際のスマートサイトMVPが動きます。専門知識は不要です。所要時間は10分ほどです。

目次 0. 事前準備（PowerShellを開く） 1. Node.jsをインストール 2. プロジェクトフォルダに移動 3. 必要なライブラリをインストール 4. ローカル起動 5. Claude Codeで編集する場合 困ったときは

0事前準備（PowerShellを開く）

Windowsの「スタート」メニューを押し、「PowerShell」と入力してEnterで開いてください。 黒い（または青い）画面が出たら準備完了です。

PowerShellは、文字コマンドでパソコンに指示を出すための窓です。これから出てくるコマンドはすべてここに貼り付けて実行します。

1Node.jsをインストール

すでにNode.jsが入っている場合はスキップしてください。確認は次のコマンドです。 

node -v

「v18」以上の数字が表示されればOKです。表示されない／古い場合は、下の手順でインストールしてください。

A
公式サイトを開く

ブラウザで https://nodejs.org/ja を開きます。

B
「LTS版」をクリックしてダウンロード

緑色のボタンの「LTS（推奨）」を押してファイルをダウンロードします。

C
ダウンロードしたファイルを実行してインストール

「次へ」を押し続けるだけでインストールは完了します。

D
PowerShellを一度閉じて、もう一度開く

その後、もう一度 node -v を実行し、バージョンが表示されればOKです。

2プロジェクトフォルダに移動

今回作ったMVPの中身は、すでにあなたのパソコンの中にあります。下のコマンドをそのままコピーしてPowerShellに貼り付けてください。 

cd "$env:USERPROFILE\Documents\Claude\Projects\次世代ホームページ\smart-site"

実行すると、画面の左側のパス（青い文字）が ...\smart-site に変わります。これでフォルダの中に入った状態になります。

左側に smart-site という文字が見えなければ、フォルダの場所が違います。エクスプローラーで「ドキュメント → Claude → Projects → 次世代ホームページ → smart-site」を確認してください。

3必要なライブラリをインストール

プロジェクトを動かすために必要な部品（ライブラリ）をまとめてダウンロードします。1〜3分かかります。 

npm install

たくさんの文字が流れますが、止まるまで待ってください。最後に added 〇〇 packages のような表示が出れば完了です。

途中で警告（黄色や黒い文字）が出ても、エラー（赤い文字）でなければ問題ありません。

4ローカル起動

いよいよ起動です。次のコマンドを実行してください。

npm run dev

数秒後に Local: http://localhost:3000 という表示が出ます。 その後、ブラウザを開いて下のアドレスにアクセスしてください。 

http://localhost:3000

スマートサイトMVPの画面が表示されれば成功です。

動作チェックリスト

  • トップに「作って終わりのHPから、賢く動くスマートサイトへ。」と表示される
  • 「新規のお客様／既存のお客様／求職中の方」を切り替えるとカードが入れ替わる
  • Smart Dataの数字がアニメーションでカウントアップする
  • Smart Topicsのカードが横に流れる
  • マップのアイコンを押すと、右側の詳細パネルが切り替わる
  • 診断の3問に答えると、結果ページが出る
  • AIナビのチップを押すと、チャットが進む
  • Before/Afterのトグルで内容が切り替わる

止め方

PowerShell上で Ctrl + C を押せばサーバーが止まります。

5Claude Codeで編集する場合

Claude Codeを使ってこのMVPを改造したい場合は、別のPowerShellをもう一つ開き、同じフォルダで以下を実行してください。 

cd "$env:USERPROFILE\Documents\Claude\Projects\次世代ホームページ\smart-site"
claude

Claude Codeが立ち上がったら、たとえば次のように指示できます。

指示の例（クリックで開く）

「Heroセクションのキャッチコピーを変えて」

「Smart Dataの数字を、自社の実績値に差し替えて」

「Smart Topicsのカードに画像エリアを追加して」

「lib/mockData.ts のデータをSupabaseから取ってくるように変更して」

!困ったときは

「npm: コマンドが認識されません」と出る

Node.jsが入っていない、もしくはPowerShellを開き直していないのが原因です。手順1のDをやり直してください。

http://localhost:3000 を開いても何も表示されない

PowerShell側で npm run dev が実行中かどうか確認してください。止まっていれば、もう一度実行します。すでに別のアプリが3000番ポートを使っている場合、自動で3001番などに切り替わるので、PowerShellの表示にしたがってください。

赤い文字でエラーが出た

そのエラーメッセージをそのままClaudeにコピーして送ってください。原因と直し方を一緒に調べます。

もう一度最初からやり直したい

smart-siteフォルダの中の node_modules フォルダを削除してから、もう一度 npm install を実行してください。
